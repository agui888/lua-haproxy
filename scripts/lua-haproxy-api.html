Script lua-haproxy-api

Somewhat RESTful API for HAProxy.

Usage:

     global
   stats socket /run/haproxy/admin.sock mode 660 level admin
   lua-load /etc/haproxy/lua-haproxy-api.lua

 frontend example
   http-request use-service lua.haproxy-api
